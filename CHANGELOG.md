# سجل الإصدارات — منصة المواقع الاستثمارية بالمدينة المنورة

يتبع هذا السجل [الترقيم الدلالي SemVer](https://semver.org/lang/ar/): **رئيسي.فرعي.إصلاح**
- **رئيسي**: تغيير يكسر التوافق (بنية التخزين، صيغ التصدير)
- **فرعي**: ميزة جديدة متوافقة
- **إصلاح**: تصحيح خلل دون ميزات

---

## [1.0.0] — 2026-07-19

الإصدار الأول المكتمل بعد اجتياز بوابات مراجعة المعادلات (33/33) وتدقيق الجودة الشامل.

### الأساس
- خريطة تفاعلية (Leaflet) بأربع خرائط أساس، محمّلة مسبقاً بـ 1015 قطعة من بيانات أمانة منطقة المدينة المنورة.
- استيراد KML/KMZ/GeoJSON — ملفات متعددة دفعة واحدة، كأراضٍ أو كمعالم، مع خيارات التصنيف والنوع الهندسي.
- تصنيف تلقائي بالمساحة (50/100 ألف م²) بألوان قابلة للتعديل من مفتاح الخريطة.
- أقسام مخصصة كاملة، وبطاقة أرض ببيانات أساسية قابلة للتعديل (رقم القطعة، اسم الشارع) ومعالم تركيز متعددة الاختيار.
- معالم بأربعة أنواع هندسية: نقطة، خط (طريق)، مضلع (مشروع)، حلقة (طرق دائرية) — إدخال بالرسم على الخريطة أو بالإحداثيات يدوياً.
- تحليل المسافات: مباشرة + اتجاه عربي + مسافة الطريق وزمن الوصول (OSRM)، بحد نتائج قابل للضبط.
- إظهار/إخفاء كل معلم من جميع الجداول والخرائط والطباعة، وفلاتر فئات موحّدة متزامنة بين الصفحات.
- تقارير PDF بقالب A4 عرضي بصفحة واحدة (بيانات يميناً + خريطتا الموقع العام والمباشر يساراً) مع شريط الأمانة، وطباعة عدة مواقع في ملف واحد، وتحكم بخريطة الأساس وفلاتر التقرير قبل الطباعة، وتصدير كل خريطة صورة PNG.
- تصدير Excel/GeoJSON/KML، ونسخة احتياطية JSON كاملة.
- حفظ تلقائي فوري في IndexedDB + زر «حفظ الكل» + مزامنة لحظية بين التبويبات.
- واجهة عربية RTL كاملة بوضعين نهاري/ليلي، متجاوبة (جوال/لوحي/حاسب) بأحجام لمس ≥44px.
- خط نشر CI/CD إلى GitHub Pages يفحص الكود والبيانات قبل كل نشر.
