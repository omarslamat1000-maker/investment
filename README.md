# منصة المواقع الاستثمارية بالمدينة المنورة 🗺️

تطبيق ويب عربي متكامل (RTL) لتحليل وإدارة الأراضي داخل المدينة المنورة — يعمل بالكامل في المتصفح دون خادم خلفي، ويحفظ بياناتك محلياً في المتصفح عبر IndexedDB.

## المزايا

- **استيراد KML/KMZ**: رفع ملفات تحتوي آلاف قطع الأراضي وتحويلها إلى GeoJSON مع الحفاظ على جميع الخصائص الأصلية، ومعالجة المضلعات غير المغلقة.
- **بيانات مدمجة**: يأتي التطبيق محمّلاً مسبقاً ببيانات 1015 قطعة من ملف «الأراضي التابعة لأمانة المدينة المنورة».
- **تصنيف تلقائي بالمساحة**: أقل من 50 ألف م² / 50–100 ألف م² / أكثر من 100 ألف م²، بألوان قابلة للتخصيص من الإعدادات.
- **أقسام مخصصة**: إنشاء/تسمية/حذف أقسام، نسخ القطع إليها (قطعة واحدة في أكثر من قسم)، تسمية مخصصة داخل القسم، ملاحظات وحالة وأولوية لكل أرض، تصدير القسم إلى Excel/GeoJSON/KML.
- **معالم المدينة المنورة**: المسجد النبوي، المطار، محطة قطار الحرمين، جبل أحد، مسجد قباء، القبلتين، ميقات ذي الحليفة، الطرق الدائرية الثلاثة (كحلقات)، والمحاور الرئيسة — مع صفحة كاملة لإدارتها وإضافة معالم جديدة بالنقر على الخريطة.
- **تحليل الموقع والمسافات**: مسافة مباشرة + اتجاه جغرافي لجميع المعالم مرتبة من الأقرب، ومسافة الطريق وزمن الوصول بالسيارة عبر خدمة OSRM عند توفر الإنترنت، مع تصدير التقرير Excel أو PDF.
- **فلاتر متقدمة**: فئة المساحة، نطاق المساحة، الاسم، رقم القطعة، الحي/الشارع، القسم، الحالة، الأولوية، القرب من معلم، وغيرها — بتحديث فوري للخريطة والإحصاءات.
- **لوحة معلومات**: إحصاءات شاملة + رسوم بيانية (Chart.js).
- **أدوات خريطة**: قياس مسافة/مساحة، رسم، بحث جغرافي (Nominatim)، تحديد الموقع، ملء الشاشة، لقطة PNG، طباعة، و4 خرائط أساس (شوارع/أقمار صناعية/فاتحة/داكنة).
- **تقارير PDF**: صفحات طباعة أنيقة (تقرير عام، بطاقة أرض، تقرير مسافات) عبر «طباعة/حفظ PDF» من المتصفح — بدعم عربي كامل.
- **حفظ تلقائي + نسخ احتياطي**: كل شيء يُحفظ في IndexedDB، مع تصدير/استيراد نسخة احتياطية JSON كاملة، وفصل تام بين البيانات الأصلية وتعديلات المستخدم.
- **وضع ليلي/نهاري** وتصميم متجاوب للحاسب والأجهزة اللوحية والجوال.

## التشغيل محلياً

يفضَّل تشغيل التطبيق عبر خادم محلي بسيط (لأن بعض المتصفحات تقيّد `file://`):

```bash
# بأي من الطرق التالية داخل مجلد المشروع:
npx serve .
# أو
python -m http.server 8000
```

ثم افتح: `http://localhost:8000`

> ملاحظة: فتح `index.html` مباشرة بالنقر المزدوج يعمل أيضاً في أغلب المتصفحات لأن البيانات مدمجة كملف JS، لكن الخادم المحلي هو الأضمن.

يتطلب التطبيق اتصالاً بالإنترنت لتحميل بلاطات الخرائط والمكتبات (CDN) وخدمتي البحث الجغرافي والمسارات؛ البيانات نفسها تبقى محلية على جهازك ولا تُرسل لأي جهة.

## النشر على GitHub Pages (بخط نشر تلقائي)

المشروع مجهّز بخط CI/CD في `.github/workflows/deploy.yml`: عند كل دفعة إلى `main` تُنفَّذ الفحوصات
(صياغة كل ملفات JS + وجود الصفحات + سلامة بيانات الأراضي المدمجة)، وإن نجحت كلها يُنشر الموقع تلقائياً.

1. أنشئ المستودع وارفع المشروع (مرة واحدة):
   ```bash
   cd medina-lands
   git init
   git add .
   git commit -m "الإصدار الأول — منصة المواقع الاستثمارية بالمدينة المنورة"
   git branch -M main
   git remote add origin https://github.com/USERNAME/REPO.git
   git push -u origin main
   ```
2. من إعدادات المستودع فعّل النشر عبر Actions: **Settings ← Pages ← Source: GitHub Actions**.
3. سيعمل خط النشر تلقائياً، وبعد اكتماله يكون التطبيق متاحاً على:
   `https://USERNAME.github.io/REPO/`
4. **كل تحديث لاحق**: ادفع إلى `main` فقط — الفحص والنشر تلقائيان، وأي فشل في الفحوصات يوقف النشر ويُبقي النسخة السابقة تعمل.

### خطة التراجع (Rollback)

إن ظهرت مشكلة بعد نشر ما:
```bash
git revert HEAD      # يعكس آخر تحديث بالتاريخ محفوظاً
git push             # يعيد نشر النسخة السابقة تلقائياً عبر نفس الخط
```

### النسخ الاحتياطي للبيانات

بيانات المستخدمين (الأقسام، التعديلات، المعالم) تعيش في **متصفح كل مستخدم** (IndexedDB) لا على الخادم —
لذا النسخ الاحتياطي بيد المستخدم من داخل التطبيق: **⚙ الإعدادات ← إنشاء نسخة احتياطية (JSON)**،
والاستعادة من الزر المجاور. يُنصح بنسخة دورية قبل أي استيراد كبير أو مسح.

## تحديث البيانات المدمجة

لتحديث ملف البيانات المدمج من ملف KMZ/KML جديد:

```bash
# استخرج doc.kml من ملف KMZ (ملف KMZ هو أرشيف ZIP)، ثم:
node tools/convert-kml.js doc.kml data/lands-data.js
```

أو ببساطة استخدم زر «رفع KML/KMZ» داخل التطبيق — البيانات المرفوعة تُحفظ في المتصفح وتَحلّ محل المدمجة.
